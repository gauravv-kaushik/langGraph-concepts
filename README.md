LangGraph Concepts

⚠️ Important: Create a .env file in the root directory and add your API key:
GOOGLE_API_KEY="your_api_key_here"

📌 Overview

This project demonstrates core concepts of LangGraph, including building AI workflows, chaining LLM calls, and managing state across nodes.
It is designed to help developers understand how to structure and scale LLM-powered applications using graph-based logic.

🚀 Features

LangGraph workflow implementation
Integration with Google Generative AI (Gemini)
State management across nodes
Modular and scalable architecture
Easy-to-understand examples

🛠️ Tech Stack

Python
LangChain / LangGraph
Google Generative AI (Gemini)
dotenv

📂 Project Setup

1. Clone the repository
   git clone https://github.com/your-username/langgraph_concepts.git
   cd langgraph_concepts

2. Create virtual environment
   python -m venv env
   source env/bin/activate # Mac/Linux
   env\Scripts\activate # Windows

3. Install dependencies
   pip install -r requirements.txt

4. Setup environment variables

   Create a .env file and add:
   GOOGLE_API_KEY="your_api_key_here"

▶️ Run the Project

python filename.py
